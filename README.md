# ELK Stack with Zeek for Modbus Security Monitoring

A complete setup for monitoring and analyzing Modbus/ICS traffic using the ELK stack (Elasticsearch, Kibana) with Zeek network security monitor and Filebeat.

## 🏗️ Architecture
Zeek → Modbus Logs → Filebeat → Elasticsearch → Kibana

## 📁 Project Structure
elk-zeek-modbus/
├── docker-compose.yml          # ELK stack configuration
├── configs/                    # Zeek and Filebeat configurations
├── docs/                       # Setup and Kibana guides
└── scripts/                    # Utilities and test scripts
